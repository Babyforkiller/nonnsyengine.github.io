Политика конфиденциальности

Дата последнего обновления: 18 февраля 2026 г.

1. Общие положения

1.1. Настоящая Политика конфиденциальности (далее — «Политика») определяет порядок обработки и защиты персональных данных пользователей Telegram-бота NONNSY ENGINE (далее — «Сервис», «мы»).

1.2. Используя Сервис, вы соглашаетесь с условиями настоящей Политики.

1.3. Администрация Сервиса обязуется соблюдать конфиденциальность персональных данных пользователей.

2. Собираемые данные

2.1. При использовании Сервиса мы можем собирать следующие данные:

  • Идентификатор пользователя Telegram (User ID)
  • Имя пользователя Telegram (username)
  • Информация о подключённых аккаунтах (номера телефонов, API ID, API Hash)
  • Настройки использования Сервиса
  • Данные о транзакциях и оплате
  • Технические данные (IP-адрес, тип устройства, версия браузера)

2.2. Мы не собираем и не храним содержимое сообщений, отправляемых через Сервис.

3. Цели обработки данных

3.1. Персональные данные обрабатываются в следующих целях:

  • Предоставление доступа к функционалу Сервиса
  • Обработка платежей и управление подписками
  • Обеспечение безопасности и предотвращение мошенничества
  • Улучшение качества Сервиса
  • Оказание технической поддержки
  • Соблюдение требований законодательства

4. Способы обработки данных

4.1. Обработка персональных данных осуществляется с использованием средств автоматизации и без использования таких средств.

4.2. Данные хранятся в зашифрованном виде на защищённых серверах.

4.3. Мы применяем современные методы защиты данных от несанкционированного доступа.

5. Передача данных третьим лицам

5.1. Мы не передаём персональные данные третьим лицам, за исключением следующих случаев:

  • Пользователь дал согласие на передачу данных
  • Передача необходима для предоставления услуг (например, платёжные системы)
  • Передача требуется по требованию законодательства или уполномоченных органов

5.2. При передаче данных третьим лицам мы обеспечиваем их защиту и соблюдение конфиденциальности.

6. Защита данных

6.1. Мы применяем технические и организационные меры для защиты персональных данных от несанкционированного доступа, уничтожения, изменения или распространения.

6.2. Все данные передаются по защищённым каналам связи с использованием современных протоколов шифрования.

6.3. Доступ к персональным данным имеют только уполномоченные сотрудники, которые обязаны соблюдать конфиденциальность.

7. Права пользователей

7.1. Пользователь имеет право:

  • Получать информацию о своих персональных данных
  • Требовать исправления неточных данных
  • Требовать удаления своих персональных данных
  • Отозвать согласие на обработку данных
  • Ограничить обработку данных

7.2. Для реализации своих прав пользователь может обратиться к нам через контакты, указанные в разделе «Контакты».

8. Хранение данных

8.1. Персональные данные хранятся в течение срока, необходимого для достижения целей обработки.

8.2. После прекращения использования Сервиса данные могут храниться в течение срока, установленного законодательством.

8.3. Пользователь может запросить удаление своих данных в любое время.

9. Использование файлов cookie

9.1. Веб-сайт Сервиса может использовать файлы cookie для улучшения пользовательского опыта.

9.2. Использование сайта подразумевает согласие с использованием cookie.

9.3. Пользователь может отключить использование cookie в настройках браузера.

10. Изменение Политики

10.1. Мы имеем право изменять настоящую Политику в любое время.

10.2. Изменения вступают в силу с момента публикации новой версии Политики на сайте Сервиса.

10.3. Продолжение использования Сервиса после изменений означает согласие с новой Политикой.

11. Контакты

По всем вопросам, связанным с обработкой персональных данных, обращайтесь:

  • Telegram-бот: @Nonnsy_bot
  • Администратор / поддержка: @nonnsy
  • Отзывы: @nonnsybot_rep
