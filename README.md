# NONNSY ENGINE

**Мощный Telegram-бот для автоматизации рассылок.**

Сайт-витрина проекта: [NONNSY ENGINE](https://babyforkiller.github.io/nonnsyengine.github.io/)

## О проекте

NONNSY ENGINE — инструмент для автоматизации рассылок в Telegram:

- До 3 аккаунтов одновременно  
- Массовые рассылки с гибкими настройками  
- Автоответчик и умная защита от блокировок  
- Индивидуальное расписание для каждого аккаунта
